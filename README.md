## 🌍 Research Focus

I build **sovereign, zero-dependency technology** that puts individuals in control of their tools, data, and cognitive infrastructure.

**Principles:**
- **Individual Freedom** — Tools run on hardware you possess
- **Global Safety** — Safety systems that cannot be disabled
- **Maximum Uptime** — Offline-first, online-optional
- **Privacy for All** — Privacy by architecture, not policy
